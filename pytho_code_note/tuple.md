튜플 생성

```python
tuple_1 = (1,2)
a = 1,               # 이 경우 튜플 대입(우변 값을 튜플로 인식, 이후 좌변에 대입)
print(a, type(a))     # (1,) <class tuple'>
```

슬라이싱

```python
a = [1, 2, 3, 4]
a.[:2]                # [1, 2] ,   [0:2]와 같음
a.[0:4:2]            # [1, 3],     [0:4]에서 2칸 간격으로
a.[::]                # [1, 2, 3, 4], all index
a.[::-1]              # [4, 3, 2, 1]
```
