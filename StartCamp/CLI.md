# CLI INTRO

명령 줄 인터페이스

- GUI는 CLI에 비해 사용하기 쉽지만, 단계가 많고 컴퓨터의 성능을 더 많이 소모

- 수 많은 서버/개발 시스템이 CLI를 사용



### 각종 명령어들

#### 파일 생성

**touch**

```bash
$ touch 파일명
```

---

#### 파일 목록 확인

**ls**

```bash
$ ls
```

**ls**를 하면 현재 CLI 창이 작업중인 폴더의 파일들을 확인

**ls -a**를 하면 상위파일을 볼 수 있음 (숨김 파일도)

**ls -l**를 하면 파일들의 상세 속성을 볼 수 있음

---

#### 파일 열기

**start**

---

#### 파일 위치 확인

**pwd**

```bash
$ pwd
```

---

#### 폴더 만들기

**mkdir**	make directory

```bash
$ mkdir 폴더명
```

---

#### 작업 중인 디렉토리 변경

**cd**

```bash
$ cd my_directory/
```

현재 작업중인 cli의 디렉토리를 변경

**cd ..** 하면 상위 폴더로 이동

but 상위, 하위로만 이동 가능?

tab 눌러서 디렉토리명 자동완성 가능

---

#### 파일 삭제

**rm**

```bash
$ rm trash.md
```

**rm -r** 하면 디렉토리 삭제도 가능

---



**ctrl + l** 	화면 아래로 내림

**../** 			현재 작업중인 폴더의 부모 폴더
