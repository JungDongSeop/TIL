이차원 배열 or ArrayList 정렬할 때는

```java
        Comparator<int[]> comparator = new Comparator<int[]>() {
            public int compare(int[] arr1, int[] arr2) {
                if (arr1[0] == arr2[0]) {
                    return arr1[1] - arr2[1];
                } else {
                    return arr1[0] - arr2[0];
                }
            }
        };

        Collections.sort(targets, comparator);
```
